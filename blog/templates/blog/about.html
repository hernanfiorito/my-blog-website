About this blog

This is my first full stack web project, developed using the Django framework!

Applying the knowledge and skills I learnt in a Django bootcamp, I implemented many useful features for a blog site, including:

  • Login/Logoff system for blog staff (admin and registered users authorized to post).
  • Create and save a post draft.
  • Apply styles to comments and post text body with the Medium Editor inline toolbar.
  • Edit or delete unpublished post drafts (as a blog staff).
  • Publish a draft as a new post.
  • Edit or delete a published post (as a blog staff).
  • Add a comment in a post as a visitor.
  • Approve or disapprove comments (as a blog staff).
  • Show all published posts on the homepage as a list, including title, publish date and amount of comments in the post.

Source code is publicly available in my Github repository.
